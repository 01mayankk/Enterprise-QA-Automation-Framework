# Reports Directory

This directory stores HTML execution reports generated by Pytest.

## Contents
* `report.html`: The interactive HTML report containing pass/fail metrics, step details, execution logs, and embedded screenshots of failed steps.
* `execution_summary.md`: A markdown execution summary outlining the date, browser, environment, and pass rates.

## Usage
Reports are generated automatically by running the test suite with `pytest --html=reports/report.html`.
